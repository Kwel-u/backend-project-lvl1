# nodejs-package

## Installation
### From NPM
```sh
$ npm i backend-project-kwel_u
```
### From GitHub
```sh
$ git clone https://github.com/Kwel-u/backend-project-lvl1.git
$ make link
```

### Remember to install dependencies
```sh
$ npm install
```

## Usage
```sh
$ brain-even
```

## Available commands
```sh
$ make lint
$ make unlink
$ brain-even
$ brain-calc
$ brain-gcd
$ brain-progression
$ brain-prime
$ brain-games
```
